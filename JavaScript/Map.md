자바스크립트의 Map은 키-값 쌍을 저장하는 데이터 구조이다.
### Map의 주요 특징

1. 키-값 쌍 저장: Map은 키와 그와 연결된 값을 쌍으로 저장한다.
2. 키의 유일성: 각 키는 Map 내에서 유일해야 한다.
3. **다양한 키 타입**: 문자열뿐만 아니라 객체, 함수 등 모든 타입을 키로 사용할 수 있다.
4. 삽입 순서 유지: Map은 요소들의 삽입 순서를 기억한다.
### Map vs 객체

일반 객체와 유사해 보인다.

5. **키 타입**: 객체의 키는 문자열이나 심볼만 가능하지만, Map은 모든 값을 키로 사용할 수 있다.
6. 크기 확인: Map은 size 프로퍼티로 쉽게 크기를 확인할 수 있다.
7. 순회: Map은 for…of 루프나 forEach 메서드로 쉽게 순회할 수 있다.

Map은 키-값 쌍의 빈번한 추가/제거가 필요한 경우나 키의 타입이 미리 알려지지 않은 경우에 특히 유용하다. 또한 많은 양의 데이터를 다룰 때 성능상의 이점도 있다.

Map은 내부적으로 해시 테이블을 사용하여 키를 관리하므로, 데이터 조회 및 삭제 성능이 객체보다 더 우수한 경우가 많다.
