# Disclosure Pattern

'Disclosure Pattern'은 웹 접근성을 위한 디자인 패턴이다. 사용자가 원하는 경우에만 추가 정보가 보이도록 하는 방식으로, 웹 페이지의 정보를 숨기거나 보여주는 방식으로 제어한다.
